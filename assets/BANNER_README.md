# 🎨 TON MANDS Banner

Banner untuk TON MANDS Telegram Bot & Mini App

## Spesifikasi
- **Size**: 1280x720 (16:9)
- **Theme**: Blue Neon Crypto
- **Logo**: TON MANDS
- **Tagline**: BUILD • HOLD • GROW

## Penggunaan

### 1. BotFather Profile Photo
```
/mybots → Pilih Bot → Edit Bot Profile Photo → Upload banner
```

### 2. Mini App Photo
```
/newapp → Upload banner (640x360)
```

### 3. Channel Photo
```
Create Channel → Upload banner
```

## Files

- `banner.jpg` - Main banner (1280x720)
- `banner_mini.png` - Mini App version (640x360)
- `logo.png` - Logo only (transparent)
